# Listme-bouncer

## Requirements

- NodeJs
- Yarn
- Expo
- XCode (iOS)
- Android Studio (Android)

## Setup

```
yarn install
```

## To run

iOS

```
yarn ios
```

Android

```
yarn android
```

Web

```
yarn web
```

## Build

```
expo build:android
expo build:android -t app-bundle
or
expo build:android -t apk

expo build:ios
```
